# MineHaus Public Issue Tracker

Welcome to the public MineHaus issue tracker.

This is where players can report bugs, suggest improvements, and follow the status of known issues across the network.

Please do not post:
- IP addresses
- Private messages
- Personal information
- Exploits or security issues
- Staff reports or appeals

For urgent support, player reports, appeals, or private matters, please use the MineHaus Discord help desk.
